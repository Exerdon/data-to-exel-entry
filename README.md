# Cardholder Expense Submission Portal (CESP)
helps to submit expenses allocation and receipts
in a single page and then can be viewed on a excel style sheet 
as it creates a csv file after one is done adding expenses


## Overview
CESP is a web-based expense submission portal designed to simplify the process of cardholder expense management. The system provides an organized interface for submitting expense details efficiently.

## Features
- Expense submission form
- User-friendly interface
- Structured data input
- Responsive design
- Form validation

## Problem It Solves
Managing expense submissions manually can be inefficient and error-prone. This portal provides a centralized and structured method for submitting cardholder expenses.

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Workflow
1. User enters expense details
2. Information is validated
3. Submission is processed
